# SpikingRx-on-OAI 專題最終流程指南

以下為完整的最終實驗流程指令，從 OAI NR Softmodem 啟動、UE 連線、bundle 建立、SpikingRx inference 到結果檢查與視覺化，所有中間步驟也包含在內，方便第一次接觸的人完整理解整個流程。

---

## 1️⃣ 清理與準備環境

```bash
# 刪除先前生成的 raw dump 與 bundle
find ~/SpikingRx-on-OAI/spx_records/raw -mindepth 1 -delete
find ~/SpikingRx-on-OAI/spx_records/bundle -mindepth 1 -delete
```

## 2️⃣ 設定噪聲功率

```bash
# 設定 noise power，例如 NP = -4.68 dB
NP=-4.68
sed -Ei 's/noise_power_dB[[:space:]]*=.*/noise_power_dB = '${NP}';/' \
~/openairinterface5g/targets/PROJECTS/GENERIC-NR-5GC/CONF/channelmod_rfsimu.conf
```

## 3️⃣ 啟動 gNB（NR Softmodem）

```bash
cd ~/openairinterface5g/cmake_targets/ran_build/build
./nr-softmodem \
  --rfsim \
  --phy-test \
  --gNBs.[0].min_rxtxtime 6 \
  -O ../../../targets/PROJECTS/GENERIC-NR-5GC/CONF/gnb.sa.band78.fr1.106PRB.usrpb210.conf \
  2>&1 | tee /tmp/gnb.log
```

## 4️⃣ 啟動 UE（NR UE Softmodem）

```bash
cd ~/openairinterface5g/cmake_targets/ran_build/build
./nr-uesoftmodem \
  --rfsim \
  --phy-test \
  -r 106 --numerology 1 --band 78 -C 3619200000 --ssb 516 \
  -O ../../../targets/PROJECTS/GENERIC-NR-5GC/CONF/ue.conf \
  2>&1 | tee /tmp/ue.log
```

## 5️⃣ 編譯 SpikingRx 相關工具

## 8️⃣ 檢查設定與結果

```bash
python3 - <<'PY'
import json
from pathlib import Path
import re

# 1. 檢查 noise_power_dB
conf = Path('~/openairinterface5g/targets/PROJECTS/GENERIC-NR-5GC/CONF/channelmod_rfsimu.conf').expanduser()
text = conf.read_text()
vals = list(map(float, re.findall(r'noise_power_dB\s*=\s*(-?\d+(?:\.\d+)?)', text)))
print('=== CHECK noise_power_dB ===')
if vals:
    uniq = sorted(set(vals))
    print('values =', uniq)
    if len(uniq) == 1:
        print(f'OK: all noise_power_dB = {uniq[0]}')
    else:
        print('WARNING: inconsistent values!', vals)
else:
    print('ERROR: no noise_power_dB found')

# 2. 檢查 BER
summary = Path('~/SpikingRx-on-OAI/spx_records/bundle/spikingrx_batch_summary.json').expanduser()
s = json.loads(summary.read_text())
bad = [r for r in s['results'] if r.get('ber_pred', 0) > 0]
print('\n=== CHECK BER ===')
print('total =', len(s['results']))
print('nonzero =', len(bad))
if bad:
    print('\nexamples:')
    for r in bad[:5]:
        print(r['bundle'], r['ber_pred'])
else:
    print('OK: ALL BER = 0')
PY
```
