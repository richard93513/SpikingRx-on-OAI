## (A) 編譯 OAI（gNB + UE）

1. 清除舊 build（避免版本衝突）
```
cd ~/openairinterface5g/cmake_targets/ran_build/build
rm -rf CMakeCache.txt CMakeFiles
```

2. 重新編譯（SIMU / RFsim 模式）
```
cd ~/openairinterface5g/cmake_targets
./build_oai --gNB --nrUE -w SIMU
```

## ✅ (B) 啟動 gNB（RFsim 模式，最新版、你現在用的）

在 Terminal 1：
```
cd ~/openairinterface5g/cmake_targets/ran_build/build
./nr-softmodem \
  --rfsim \
  --phy-test \
  --gNBs.[0].min_rxtxtime 6 \
  -O ../../../targets/PROJECTS/GENERIC-NR-5GC/CONF/gnb.sa.band78.fr1.106PRB.usrpb210.conf \
```

## ✅ (D) Raw Dump（Fullgrid / TXBITS / LDPC / OAI LLR）

OAI patch 會輸出四種 raw 檔案到：

```
~/SpikingRx-on-OAI/spx_records/raw/
```

範例：
```
fXXXX_sYY_fullgrid_idxNNNNNN.bin
fXXXX_sYY_txbits_idxNNNNNN_rntiXXXX.bin
fXXXX_sYY_ldpc_cfg_idxNNNNNN.txt
fXXXX_sYY_oai_llr_idxNNNNNN.bin
```

確認是否正常 dump：
```
ls -ltr ~/SpikingRx-on-OAI/spx_records/raw | tail
```

## ✅ (E) Raw → Bundle（自動配對 fullgrid / txbits / ldpc / OAI LLR）

執行：
```
cd ~/SpikingRx-on-OAI
source .venv/bin/activate
python src/tools/bundle_records.py
```


輸出範例：
```
spx_records/bundle/f0424_s05/
├── fullgrid.bin
├── txbits.bin
├── ldpc_cfg.txt
├── oai_llr.bin
└── meta.json
```

每個 bundle 代表一筆完整 PDSCH 傳輸。

---

## ✅ (F) 新版 SpikingRx 訓練（用 OAI LLR 當 label）

執行：
```
cd ~/SpikingRx-on-OAI
source .venv/bin/activate
PYTHONPATH=src python3 src/train/train_spikingrx_oai.py
```

訓練資料：
- `x` = fullgrid.bin → (T=3, H=32, W=32)
- `y` = oai_llr.bin（長度 G = 14400）

模型目標：  
模仿 OAI 的 demapper，使 SpikingRx 輸出與 OAI LLR 越接近越好。

最佳權重會存到：
```
src/train/best_spikingrx_model.pth
```

---

## ✅ (G) 新版 Inference + LDPC 解碼 + BER（整合版）

執行：
```
cd ~/SpikingRx-on-OAI && source .venv/bin/activate && PYTHONPATH=src python3 src/inference/batch_inference_on_bundle.py --rmunmatch ~/openairinterface5g/cmake_targets/ran_build/build/rmunmatch_spx --ldpctest ~/openairinterface5g/cmake_targets/ran_build/build/ldpctest_spx --also_run_oai_baseline
```
對每個 bundle 自動完成：

1. **fullgrid → SpikingRx inference**
2. 產生：
   - `infer_llr_float.npy`
   - `infer_llr_int8.bin`
3. **呼叫 OAI LDPC Decoder：**
```
ldpctest_spx infer_llr_int8.bin ldpc_cfg.txt decoded_bits.bin
```
4. 解出 `decoded_bits.bin`
5. 比對 `txbits.bin` → 計算 BER
6. 在 terminal 顯示每筆 BER & 最終總表

輸出檔案（bundle 資料夾）：
```
fullgrid.bin
txbits.bin
ldpc_cfg.txt
oai_llr.bin
meta.json

infer_llr_float.npy
infer_llr_int8.bin
decoded_bits.bin
infer_meta.json
```

- 計算 BER
- 更新結果
