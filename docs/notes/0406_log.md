## 2️⃣ SNR 計算（關鍵轉換）

你不是直接用 noise dB，而是：

👉 用 channel estimate 算：

```
SNR = E[|H|^2] / nvar
```

也就是：

```
        E[|H|^2]
SNR = ------------
           nvar
```

並輸出：

```
noise_to_snr_summary.csv
```

### 位置：

```
snapshots_snr/noise_to_snr_summary.csv
```

👉 之後所有圖都是：

```
x 軸 = snr_top10_db
```
