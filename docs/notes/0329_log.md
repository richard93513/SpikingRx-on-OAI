## 3️⃣ 你遇到的問題

### ❌ 問題 1：dB 不等於 SNR

你現在用的是：

```text
noise_power_dB（RFsim設定）
```

但：

👉 這不是 SNR

因為：

- signal power 不一定固定
- FFT / scaling / equalization 都會影響
- OAI 沒直接輸出 SNR

---

### ❌ 問題 2：用 LLR 推 SNR → 完全錯

你試過：

```text
mean(llr)^2 / var(llr)
```

結果：

```text
SNR ≈ -48 dB（全部一樣）
```

👉 原因：

- LLR 是經過 demapper + scaling
- 已經不是物理訊號

---

### ❌ 問題 3：用 DMRS + fullgrid 算 → 也錯

你後來用：

```text
Y - H*X
```

但結果：

```text
SNR ≈ 44 dB（全部一樣）
```

👉 原因：

🔴 關鍵錯誤：

你用的是：

```text
同一個 Y 去 estimate H
再用 H 回推 Y
```

→ noise 被吸進 H

👉 殘差幾乎 0 → SNR 假高

---

### ❌ 問題 4：eqsymb / chestext 不能用

你確認過：

👉 它們是：

```text
data RE supervision
```

不是：

```text
DMRS RE
```

👉 無法算 channel SNR
